# اعتمر — Umrah Booking Platform (Bilingual AR/EN)

موقع ثنائي اللغة (عربي/انجليزي) لمنصة **اعتمر**، وهي منصة متخصصة في خدمات العمرة تربط بين المؤدين والموكلين مع إمكانية الاستعلام عن الطلبات وتحميل التطبيق.

## 🛠️ Tech Stack

- **HTML5 & CSS3** — هيكل ثنائي اللغة RTL/LTR وتنسيق مخصص
- **Bootstrap 5.0 RTL** — تخطيط متجاوب من اليمين لليسار (AR)
- **Bootstrap 5.3 LTR** — تخطيط متجاوب من اليسار لليمين (EN)
- **Font Awesome 5.8** — مجموعة الأيقونات
- **Reem Kufi + Somar** — خطوط عربية (Google Fonts + custom)
- **Vanilla JavaScript** — قائمة موبايل، مؤشرات Carousel الدائرية

## 📄 Pages

- `index.html` — الصفحة الرئيسية (عربي RTL)
- `indexen.html` — Home page (English LTR)
- `inquire.html` — صفحة الطلبات
- `login.html` — صفحة تسجيل الدخول
- `articles.html` — صفحة المقالات

## ✨ Features

- **Navbar عربي** — شعار مزدوج، روابط (الرئيسية، الطلبات، المقالات)، dropdown اختيار اللغة (AR/EN مع أعلام)، زر تسجيل/دخول
- **Hero الكعبة** — صورة كاملة العرض للكعبة مع بطاقة استعلام عن العمرة (إدخال رقم الطلب + زر استعلام)
- **Bootstrap Fade Carousel** — 5 شرائح إخبارية بتأثير fade مع مؤشرات دوائر رقمية مخصصة (1-5) وأحداث JS لتحديثها
- **قسم الأخبار والإعلانات** — تخطيط عمودين: Carousel أبرز الأخبار (يسار) + شبكة 2×2 إعلانات (يمين)
- **قسم الأخبار** — 3 بطاقات أخبار أفقية بصورة + تاريخ + عنوان + "شاهد المزيد"
- **قسم الأحداث والفعاليات** — بطاقة "لا توجد فاعليات حالية"
- **Footer** — شعار + روابط سريعة (عن اعتمر، اتصل بنا، FAQ، سياسة الخصوصية) + تحميل التطبيق (المؤدي والموكل) بأيقونات Google Play وApple Store + روابط سوشيال ميديا (فيسبوك، انستقرام، تويتر، لينكد ان)
- **نسخة انجليزية كاملة** — `indexen.html` بنفس الهيكل LTR
- تصميم متجاوب بالكامل عبر جميع أحجام الشاشات
