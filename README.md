# اعتمر — Umrah Booking Platform (Bilingual AR/EN)

⚠️ This template was originally created by Mohammed Sabry credit is required: github.com/Mohammedsabry13

موقع ثنائي اللغة (عربي/انجليزي) لمنصة اعتمر، وهي منصة مخصصة في خدمات العمرة تربط بين المؤدين والوكيلين مع إمكانية الاستعمال على التطبيق وتحميل التطبيق.

## 💪 Tech Stack

- **HTML5 & CSS3** — هيكل ثنائي اللغة RTL/LTR وتنسيق مخصص
- **Bootstrap 5.0 RTL** — تطوير متجاوب للمشاريع اليسار (AR)
- **Bootstrap 5.3 LTR** — تطوير متجاوب للمشاريع اليمين (EN)
- **Font Awesome 5.8** — مجموعة الأيقونات
- **Reem Kufi + Somar** — خط ثنائي العربية (Google Fonts + custom)
- **Vanilla JavaScript** — واجهة موقعية موبايل، مؤشرات Carousel الدائرية

## 📄 Pages

- `index.html` — الصفحة الرئيسية (عربي RTL)
- `indexen.html` — Home page (English LTR)
- `inquire.html` — صفحة الطلبات
- `login.html` — صفحة تسجيل الدخول
- `articles.html` — صفحة المقالات

## ✨ Features

- **Navbar عربي** — شعار مزودوج، روابط (الرئيسية، الطلبات، المقـالات)؛ dropdown اختيار اللغة (AR/EN مع علامات)؛ زر تسجيل/دخول
- **Hero الكعبة** — صورة كاملة العرض للكعبة مع بظاقة استعلام عن العمرة (ادخل رقم الطلب + زر استعلام)
- **Bootstrap Fade Carousel** — 5 شرائح إخبارية بتأثير fade مع مؤشرات دوائر رقمية مخصصة (1-5) وأحد JS لتحديدها
- **قسم الاخبار والاعلانات** — تصميم مودرن: Carousel أبرز الأخبار (ياسار) + شعبة 2×2 إعلانات (يمين)
- **قسم الأخبار** — 3 بطاقات أخبار افقية بصورة واضحة + تاريخ + عنوان + "شاهد المزيد"
- **قسم الأحداث والفعلـيات** — بطاقة "لا توجـد فعاليات حالية"
- **Footer** — شعار + روابط سريعة (عن اعتمر، اتصل بنا، FAQ، سياسة الخصوصية) + تحميل التطبيق (المؤدي والمـوكليين) بأيقونات Google Play وApple Store + روابط سوشيال ميديا (فيسبوك، انستقرام، تويتر، لينكدان)
- **نسخة إنجليزية كاملة** — `indexen.html` بنفس الهيكل LTR
- **تـميم متجاوب بالكامل على الكامِل عربي**
